# DevGram — скрытые подарки

Список скрытых подарков Telegram, которые DevGram показывает в окне «Подарить подарок» сверху.

## Как добавить подарок

Открой `gifts.json` и добавь объект в массив `gifts`:

```json
{ "id": 5974210632977745012, "emoji": "🧸" }
```

- `id` — gift_id подарка (обязательно).
- `emoji` — эмодзи для подписи (необязательно).

Цену, название и фото клиент подтянет с сервера сам по `id`. После коммита изменения появятся в приложении (список обновляется при открытии окна подарков).
