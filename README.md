# DevGram — скрытые подарки

Список скрытых подарков Telegram, которые DevGram показывает в окне «Подарить подарок» сверху.

## Как добавить подарок

Открой `gifts.json` и добавь объект в массив `gifts`:

```json
{ "id": 5974210632977745012, "stars": 50, "sticker_number": 8 }
```

- `id` — gift_id подарка.
- `stars` — цена в звёздах (сколько реально спишется).
- `sticker_number` — номер стикера в паке, начиная с нуля.
- `pack` — можно указать свой стикерпак для конкретного подарка; если не указан, берётся общий из поля `pack` вверху файла.

После коммита изменения появятся в приложении при следующем открытии окна подарков.
