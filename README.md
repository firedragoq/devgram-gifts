# DevGram — скрытые подарки

Список скрытых подарков Telegram, которые DevGram показывает в окне «Подарить подарок» сверху.

## Как добавить подарок

Открой `gifts.json` и добавь объект в массив `gifts`:

```json
{ "gift_id": 5974210632977745012, "sticker_id": 5397971251878732060, "price": 15 }
```

- `gift_id` — id подарка (в пересланном сообщении это `code`).
- `sticker_id` — document_id анимированного стикера (в сообщении это `emoji-id` у эмодзи).
- `price` — цена в звёздах. Если поставить `0`, клиент подтянет цену с сервера сам.

После коммита изменения появятся в приложении при следующем открытии окна подарков.
